# BookStore
<br>
<h1 align="center"> BookStore </h1>
<p align="center"> Книжная лавка</p>

***В проекте используются следующие технологии:***

    - .Net 7.0
    - AspNetCore.OpenApi 7.0.4
    - EntityFrameworkCore 7.0.7
    - База данных MSSQLLocalDB

***Для запуска проекта необходимо:***

    1. Клонировать репозиторий на свой компьютер. 
    2. Выполнить команду для создания базы данных: Update-DataBase.
    3. Заполнить базу данных значениями (можно использовать готовые запросы из файла sql.txt в корне проекта).
    4. Запустить проект.
